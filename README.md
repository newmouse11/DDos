## DDos
# 스크립스 실행시에는 반드시 본인의 서버 또는 본인의 호스팅 주소를 사용해 주시기 바랍니다.


 사용으로 인한 피해 및 손실, 법적 피해 보상은 모두 공격자(스크립트 실행자) 에게 있음을 숙지 하였음을 인지한 상태로 실행하십시오.

 
 이 코드 중 대부분은 Gemini Ai 로 제작된 코드 입니다.

 
 이 코드는 오픈소스 이며 "피해의 목적" 이 아닌 "테스트" 용도로 만들어 졌습니다.

 일부 코드에선 stop 을 터미널에 입력하면, 코드의 실행이 중단 됩니다.
